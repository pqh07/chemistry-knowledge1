大学化学电化学核心知识点汇总
电化学是大学无机化学、物理化学核心分支，主要研究电能与化学能相互转化的规律，核心分为原电池（化学能→电能）和电解池（电能→化学能）两大体系，贯穿氧化还原反应、热力学、动力学核心考点。本知识点适配大学本科通用化学教材，兼顾基础原理、公式定律与应用考点。
一、电化学基础概念
1.1 氧化还原与电极本质
电化学的核心是自发/非自发氧化还原反应的电子定向转移。区别于普通氧化还原反应的无序电子转移，电化学体系中电子通过外电路定向移动，离子通过电解质溶液迁移，实现能量转化。
- 氧化反应（阳极通用定义）：物质失去电子，化合价升高，发生氧化，是还原剂
- 还原反应（阴极通用定义）：物质得到电子，化合价降低，发生还原，是氧化剂
- 通用电极判断规则（适用于所有电化学装置）：阳极=氧化极，阴极=还原极

1.2 电解质与电化学体系构成
完整电化学体系三要素：两个电极、电解质溶液/熔融电解质、闭合回路。
- 电解质：在水溶液或熔融状态下能电离出自由移动离子的化合物，依靠离子导电（区别于金属的电子导电）
- 离子迁移规律：原电池中，阳离子向正极移动，阴离子向负极移动；电解池中，阳离子向阴极移动，阴离子向阳极移动
- 盐桥：通常为含KCl饱和溶液的琼脂凝胶，作用是连通两个半电池、平衡电荷、消除液接电势，保证电路闭合，不参与电极反应
二、原电池与电极电势
2.1 原电池工作原理
原电池是将自发氧化还原反应的化学能直接转化为电能的装置，由两个半电池组成，每个半电池包含一组电极与对应的电解质体系，分别发生氧化、还原半反应。
原电池书写规则（国际通用）：
1. 负极（氧化极）写左侧，正极（还原极）写右侧
2. 单竖线「|」表示相界面（电极与溶液、溶液与沉淀等）
3. 双竖线「||」表示盐桥，分隔两个半电池体系
4. 标注物质状态（s、l、g、aq）、浓度、分压
示例：铜锌原电池（丹尼尔电池）
负极（Zn）：$$\mathrm{Zn(s) - 2e^- = Zn^{2+}(aq)}$$
正极（Cu）：$$\mathrm{Cu^{2+}(aq) + 2e^- = Cu(s)}$$
电池总反应：$$\mathrm{Zn(s) + Cu^{2+}(aq) = Zn^{2+}(aq) + Cu(s)}$$
电池符号：$$\mathrm{Zn(s)|Zn^{2+}(aq)||Cu^{2+}(aq)|Cu(s)}$$
2.2 电极电势与标准电极电势
2.2.1 基本定义
电极电势（φ）：电极与电解质溶液之间的电势差，是衡量电极得失电子能力的物理量，电势越高，氧化态物质得电子能力越强（氧化性越强）；电势越低，还原态物质失电子能力越强（还原性越强）。
标准电极电势（φ⊖）：标准状态下（温度298.15K、离子浓度1mol·L⁻¹、气体分压100kPa、纯固体纯液体）的电极电势。
规定：标准氢电极（SHE）的电极电势为0.000V，作为绝对基准。
标准氢电极反应：$$\mathrm{2H^+(aq) + 2e^- = H_2(g)}$$$$\varphi^\ominus(\mathrm{H^+/H_2})=0.000\mathrm{V}$$
2.2.2 核心规律
- $$\varphi^\ominus$$ 越大，对应氧化态氧化性越强，还原态还原性越弱
- $$\varphi^\ominus$$ 越小（越负），对应还原态还原性越强，氧化态氧化性越弱
- 电极电势属于强度性质，与电极反应计量数无关，不随反应式倍数变化而改变
2.3 能斯特（Nernst）方程
能斯特方程用于计算非标准状态下的电极电势，是电化学最核心公式，适配任意温度、浓度、分压条件。
2.3.1 通用公式
电极反应通式：$$\mathrm{氧化态 + ne^- = 还原态}$$
298K简化式（大学常用）：$$\varphi = \varphi^\ominus + \frac{0.0592}{n}\lg\frac{[\mathrm{氧化态}]}{[\mathrm{还原态}]}$$
通用热力学式：$$\varphi = \varphi^\ominus + \frac{RT}{nF}\ln\frac{[\mathrm{氧化态}]}{[\mathrm{还原态}]}$$
参数说明：n为电极反应转移电子数；R为气体常数8.314J·mol⁻¹·K⁻¹；F为法拉第常数96485C·mol⁻¹
2.3.2 应用要点
- 纯固体、纯液体、溶剂水不写入浓度商
- 气体代入分压（相对标准压力），离子代入相对浓度
- H⁺、OH⁻参与电极反应时，酸碱浓度会显著影响电极电势
- 浓度越大，氧化态浓度越高，电极电势越高，氧化性越强
2.4 电池电动势与热力学关系
2.4.1 电池电动势计算
电池电动势$$E$$：外电路开路时正负极电势差，$$E = \varphi_{正} - \varphi_{负}$$
标准电动势：$$E^\ominus = \varphi^\ominus_{正} - \varphi^\ominus_{负}$$
反应自发判据：$$E > 0$$，原电池反应正向自发；$$E = 0$$，反应达到平衡；$$E < 0$$，反应逆向自发（需电解驱动）
2.4.2 电动势与热力学函数关系
1. 吉布斯自由能变与电动势：$$\Delta_r G_m = -nFE$$，标准状态：$$\Delta_r G_m^\ominus = -nFE^\ominus$$
2. 平衡常数与标准电动势：$$\lg K^\ominus = \frac{nE^\ominus}{0.0592}$$（298K）
核心结论：自发反应$$\Delta G < 0、E > 0、K^\ominus > 1$$；非自发反应反之。
三、常见电极类型
3.1 金属-金属离子电极
结构：金属单质浸入对应金属离子溶液，示例：$$\mathrm{Cu^{2+}/Cu、Zn^{2+}/Zn}$$，电极反应：$$\mathrm{M^{n+} + ne^- = M}$$
3.2 气体-离子电极
结构：惰性电极（Pt、石墨）吸附气体，浸入含对应离子的溶液，无固体参与反应，示例：标准氢电极、氯电极$$\mathrm{Cl_2/Cl^-}$$
3.3 氧化还原电极
结构：惰性电极插入同一元素高低价态离子混合溶液，电极不参与反应，仅传递电子。示例：$$\mathrm{Fe^{3+}/Fe^{2+}、MnO_4^-/Mn^{2+}}$$
3.4 金属-难溶盐电极
结构：金属表面覆盖难溶盐，浸入含难溶盐阴离子的溶液，典型示例：甘汞电极、Ag-AgCl电极，常用作参比电极（电势稳定，替代标准氢电极）。
四、电解池与电解规律
4.1 电解池核心原理
电解池是消耗电能驱动非自发氧化还原反应的装置，反应能否进行由外加电压决定，核心应用：电镀、金属冶炼、电解精炼、废水处理。
4.2 放电顺序（大学通用优先级）
电解池中离子放电（得失电子）遵循优先级顺序，是判断电解产物的核心依据：
阳极放电顺序（阴离子失电子）：$$\mathrm{S^{2-} > I^- > Br^- > Cl^- > OH^- > 含氧酸根 > F^-}$$
注：活性电极（除Pt、Au外金属）作阳极时，优先电极本身失电子溶解，不考虑阴离子放电
阴极放电顺序（阳离子得电子）：金属活动性顺序逆序，$$\mathrm{Ag^+ > Fe^{3+} > Cu^{2+} > H^+（酸） > Pb^{2+} > Sn^{2+} > Fe^{2+} > Zn^{2+} > H^+（水）}$$
4.3 法拉第电解定律
定量描述电解过程中电量与产物物质的量的关系，是电解计算核心公式：
1. 电极反应产物的物质的量：$$n = \frac{Q}{nF}$$
2. 电量公式：$$Q = It$$（I为电流强度，t为通电时间）
核心含义：相同电量电解不同体系，转移电子物质的量相等，产物质量与摩尔质量、转移电子数相关。
五、金属腐蚀与防护
5.1 金属腐蚀分类
5.1.1 化学腐蚀
金属直接与干燥气体、非电解质发生氧化还原反应，无电流产生，速率较慢，如高温金属氧化、石油介质腐蚀。
5.1.2 电化学腐蚀（主要腐蚀形式）
金属与电解质溶液接触，形成微型原电池，发生自发电化学氧化，有电流产生，腐蚀速率远快于化学腐蚀，分为吸氧腐蚀和析氢腐蚀。

5.2 金属防护方法
- 涂层防护：油漆、搪瓷、金属镀层，隔绝电解质与空气
- 牺牲阳极法（原电池原理）：连接更活泼金属，活泼金属作负极被腐蚀，保护主体金属，如船舶镶嵌锌块
- 外加电流阴极保护法（电解池原理）：将被保护金属接电源负极作阴极，辅助电极接正极，强制金属不被氧化，用于管道、设备防护
- 缓蚀剂防护：添加试剂吸附在金属表面，抑制电极反应
六、重难点总结与易错点
6.1 核心重难点
1. 电极电势的影响因素：浓度、酸碱度、温度、沉淀/配合物生成（生成沉淀、配合物会降低离子浓度，改变电极电势）
2. 能斯特方程的精准计算与反应自发性判断
3. 原电池、电解池电极反应式书写，区分阴阳极、正负极
4. 电化学与热力学平衡常数、吉布斯自由能的联动计算
6.2 高频易错点
- 电极电势与反应计量数无关，不可随方程式倍数缩放
- 盐桥不参与反应，仅平衡电荷，不能改变电池总反应
- 电解活性阳极优先溶解，惰性电极才考虑离子放电
- 中性介质中优先发生吸氧腐蚀，而非析氢腐蚀
- 电池电动势为强度性质，无加和性，平衡常数有加和性
七、常用电化学常数（298K）
- 法拉第常数$$F=96485\mathrm{C·mol^{-1}}$$，近似取值$$96500\mathrm{C·mol^{-1}}$$
- 气体常数$$R=8.314\mathrm{J·mol^{-1}·K^{-1}}$$
- 常温能斯特方程系数：0.0592V
- 标准氢电极电势：$$\varphi^\ominus=0.000\mathrm{V}$$
